WC2026 Firebase Admin

Panel de administración para datos en tiempo real

📋 Instrucciones

1 Crear proyecto en Firebase Console
2 Habilitar Firestore Database (elegir modo de prueba para empezar)
3 Ir a Configuración del proyecto → Agregar app web → copiar configuración
4 Pegar la configuración abajo y conectar
5 Hacer clic en "Cargar Partidos" para poblar la base de datos
6 Copiar la config a js/firebase-config.js y cambiar FIREBASE_ENABLED = true
